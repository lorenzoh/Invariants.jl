invariant

function defined in module Invariants


			invariant(fn, title; description, validate, inputfn)

Create an invariant with name title that checks an input against fn which returns either nothing when the invariant is satisfied, or an error message when the invariant is violated.


			invariant(inv; title, description, validate, inputfn)

Wrap an invariant inv , replacing some of its attributes with the given keyword arguments.


			invariant(invs, title, combine = :all; description)

Combine multiple invariants invs logically. The type of composition dependsBy default, all invariants must be on the third argument:
